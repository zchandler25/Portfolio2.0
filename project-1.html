LunchBox is a K3S cluster of Raspberry Pi 4 housed in a 3D printed case that incorporates a network switch, PC fan, Powersupply unit and a small back up HDD. The purpose is to develop a small on-prem appliance that utilizes the cloud when workloads surpass the local resources. One use case is a semi mobile datacenter that can be installed in an RV....just gotta convince the wife.
